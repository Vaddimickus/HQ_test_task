Решение тестового задания от HQ на позицию стажёр-программист Python

Вся обработка запросов происходит в приложении courses.
Для того чтобы получить данные от API 1 нужно перейти по ссылке http://127.0.0.1:8000/api1?student_id=1 (при условии запуска на локальном хосте)
Интересующий студент указывается через параметр student_id
Для того чтобы получить данные от API 2 нужно перейти по ссылке http://127.0.0.1:8000/api2?student_id=2&product_id=1 (при условии запуска на локальном хосте)
Интересующий студент указывается через параметр student_id, курс - product_id=1
Для того чтобы получить данные от API 2 нужно перейти по ссылке http://127.0.0.1:8000/api3 (при условии запуска на локальном хосте)


Модели описаны в файле courses/models.py
Сериализаторы описаны в файле courses/serializers.py
Бэкенд API описан в файле courses/views.py
